Voice Chatbot

1. Project's Purpose

영어 교육 어플리케이션의 원어민 강사 콘텐츠 중 일부를 대체하기 위해 진행했던 프로젝트입니다.

Developed Voice Chatbot module to replace part of contents in English educating application, specifically the works performed by native English speakers.

1-1. Operating Procedure

  1. 음성 인식 모델을 사용하여 사용자의 음성을 텍스트로 전환

    2. Change user's speech into text with Speech Recognition model

  2. 대화 주제 분류 모델과 답변 생성 모델을 거쳐 텍스트 응답을 생성

    3. Generate text response by passing through Dialogue Subject Classification model and Natural Language Generation model

  3. 다시 음성 합성 모델을 사용하여 생성된 텍스트를 음성 합성 및 사용자에게 음성 답변을 반환

    4. Change the text response into speech with Speech Synthesis model and return it

2. Used Tools

OS Windows10 Linux (Ubuntu 16.04 LTS)
Programming Language Python3
Frameworks TensorFlow Kaldi Docker

3. My Role

참여 기간 2018.11 - 2019.1
수행 역할 텍스트 데이터 탐색 텍스트 데이터 전처리 대화 주제 분류 모델 개선 답변 생성 모델 개선

3-1. Searching Text Dataset

  1. 사용 목적에 적절하도록 일상적인 대화 위주의 dialogue dataset을 탐색

    2. To make our model perform in the way we want, I searched dialogue dataset which contains contents of ordinary life

  2. 많은 dataset들 중 Yanran Li의 DailyDialogue를 사용

    3. Decided to use DailyDialogue of Yanran Li

  3. 13,118개 세트의 대화 데이터들을 10개의 도메인으로 분류

    4. Classified 13,118 dialogue datas into 10 domains

3-2. Text Dataset Preprocessing

  1. 화자 별 문장 구분

    2. Seperate lines by speakers

  2. 불필요 기호 제거

    3. Eliminate unnecessary marks

  3. Stopwords 선정

    4. Decide words to be used as stopword

  4. 문장 별 token 개수 제한

    5. Restrict number of tokens per sentence

  5. Text Normalization

    6. Normalized text words to train model clearly

  6. 한빛소프트에서 서비스하는 데이터셋에 기반하여, 기존 데이터셋의 10가지 주제를 7가지로 재분류

    7. Reclassified 10 subjects of dataset into 7 to based on dataset being serviced by Hanbitsoft

3-3. Dialogue Subject Classes ClassificationModel Improvement

  1. GloVe , Fasttext , Word2Vec, TF-IDF 등의 NLU 기법 적용 및 테스트

    2. Test NLU methods with GloVe, Fasttext, Word2Vec, TF-IDF, and etc

  2. 정확한 accuracy 측정을 위해 F1 score 구현

    3. Implemented F1-Score to evaluate model better

3-4. Natural Language Generation Model Improvement

이전에 구현되어 있었던 Seq2Seq 모델에 Attention Mechanism을 추가하여 긴 문장의 input에 대한 문장 생성 능력을 향상

To Seq2Seq model which had been used, added Attention Mechanism to model to generate sentences better when length of input is long
